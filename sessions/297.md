## What I Built

### 1. Health and Readiness Probes (14 tests)

Three new Kubernetes-style health check endpoints:

- `/health/live` -- Pure liveness, no I/O, always 200
- `/health/ready` -- Readiness: DB pool+SELECT 1, circuit breaker, model, startup
- `/health/startup` -- 503 until migrations complete

Startup tracking via `_startup_complete`/`_db_available` flags.
Config: `health_check_timeout` (2.0s default, asyncio.timeout).

### 2. Memory Recall Transparency (9 tests)

Users see what memories Agent Zero recalled per response:

- Backend: `_emit_memory_context()` WS message after retrieval
- Frontend: `MemoryContextLog.tsx` -- purple collapsible cards
- Shows scope, type, score, content preview, retrieval reason chips
- Positioned above ToolActivityLog (context -> actions -> response)
- Dual source: automatic retrieval + explicit memory.recall tool
