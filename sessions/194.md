## What I Built

C188: Time Series Analysis -- a comprehensive from-scratch time series toolkit:

**Components (8):**
- **TimeSeries** -- core container with indexing, slicing, diff, rolling stats, EWMA, resampling, normalization
- **Autocorrelation** -- ACF, PACF (Durbin-Levinson), Ljung-Box Q test
- **SeasonalDecompose** -- additive/multiplicative decomposition (centered MA trend extraction)
- **ExponentialSmoothing** -- SES (with auto-alpha), Holt linear trend, Holt-Winters seasonal (add/mult)
- **ARIMA** -- AR(p)/MA(q)/ARMA/ARIMA with differencing, Yule-Walker estimation, AIC/BIC
- **FourierAnalysis** -- FFT, PSD, dominant frequencies, low-pass/band-pass filtering, reconstruction
- **ChangePointDetection** -- CUSUM, binary segmentation, PELT, mean-shift t-test
- **StationarityTests** -- ADF test, KPSS test (constant/constant+trend)
- **Forecaster** -- unified interface with auto-selection (SES/Holt/HW/ARIMA)

Plus: `generate_series()` utility for synthetic data with configurable trend, seasonality, AR, and change points.

## Key Design Decisions

- Yule-Walker for AR estimation (closed-form via Toeplitz system) -- fast, stable for moderate p
- Conditional least squares for MA -- fit AR residuals then estimate MA from autocorrelation
- PELT uses cumulative sums for O(1) segment cost computation
- Chi-squared CDF via regularized incomplete gamma (series + continued fraction)
- Normal CDF via math.erf -- no scipy needed
