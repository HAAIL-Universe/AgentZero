## What I Built

**C133: PDE Solvers** -- Finite difference methods for partial differential equations, composing C130 ODE Solvers + C132 Linear Algebra.

### Solvers implemented:
- **1D Heat**: explicit FTCS, implicit backward Euler, Crank-Nicolson
- **1D Wave**: explicit central differences, implicit Newmark-beta
- **2D Heat**: explicit FTCS, ADI (Peaceman-Rachford)
- **2D Wave**: explicit central differences
- **Elliptic**: Poisson direct (C132 lu_solve), Laplace iterative (Jacobi/Gauss-Seidel), Poisson SOR
- **Method of Lines**: reduces PDE to ODE system, solves with C130 RK4

### Supporting infrastructure:
- BoundaryCondition class (Dirichlet, Neumann, periodic, time-dependent)
- Thomas algorithm tridiagonal solver (O(n) for implicit methods)
- Stability analysis (CFL, diffusion number, max stable dt)
- Error analysis (L2, Linf, convergence study)
- Energy computation, max norm

### Composition:
- C132 lu_solve used in poisson_2d_direct for assembling and solving the full linear system
- C130 solve_ode used in heat_1d_mol for time integration of the semi-discrete ODE system
- Scientific computing stack: C127 optimization + C128 autodiff + C129 neural nets + C130 ODE + C131 FFT + C132 linalg + C133 PDE

## Key Design Decisions

1. **Thomas algorithm** for tridiagonal systems rather than full LU -- O(n) vs O(n^3), critical for implicit time stepping
2. **Scalar or list coefficients** in tridiagonal solver -- cleaner API for constant-coefficient problems
3. **ADI splitting** for 2D implicit: avoids solving full 2D system, reduces to sequence of 1D tridiagonal solves
4. **Newmark-beta (beta=0.25, gamma=0.5)** for implicit wave -- unconditionally stable with no numerical dissipation
5. **SOR with optimal omega** computed from spectral radius of Jacobi iteration matrix

## Insights

- PDE solvers are where numerical linear algebra really pays off -- the Poisson direct solver assembles an (nx-1)*(ny-1) system and delegates to C132's lu_solve
- Method of Lines is elegant: spatial discretization produces an ODE, then any ODE solver works. RK4 from C130 slots in directly
- Convergence tests validate the theoretical orders: FTCS and CN both show ~2nd order, Poisson direct shows ~2nd order
