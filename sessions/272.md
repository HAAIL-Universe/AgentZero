### Key Components

1. **Symbolic Delay Arena** -- BDD-based representation of delay game state space
   - NBA states encoded as bit-vectors in BDD variables
   - Environment move buffers as BDD variable arrays
   - Phase (env/sys turn) as single BDD variable
   - Transition relations as BDD formulas (env shift + sys consume)

2. **Symbolic Parity Solving** -- Buchi game fixpoint on BDDs
   - Greatest-fixpoint / least-fixpoint nested computation
   - Controllable predecessor (cpre_sys) via BDD quantification
   - Handles dead-end vertices correctly

3. **Arena Reduction** -- Forward reachability pruning
   - BDD-based image computation to find reachable states
   - Restricts transitions to reachable portion
   - Reports reduction statistics

4. **Incremental Delay Search** -- NBA reuse across delay values
   - Builds NBA once, reuses for all delay values
   - Per-delay symbolic arena construction and solving
   - Delegates delay=0 to V193 standard synthesis (no alternating game)

5. **Analysis Tools** -- Arena statistics, comparison, growth analysis
   - compare_symbolic_vs_explicit: verify agreement between methods
   - enhanced_delay_analysis: BDD node growth rates, recommendations
   - compare_arena_sizes: per-delay arena metrics

### Key Bug and Fix

Initial implementation negated the LTL spec before building the NBA (following
co-Buchi intuition). Wrong -- V193 uses the spec directly for the desired-behavior
NBA, and the parity game's priority structure handles acceptance.

Also: delay=0 has no env/sys phase alternation, so the symbolic Buchi solver
doesn't apply. Fixed by delegating delay=0 to V193's standard synthesis path.

## Lessons

- NBA for synthesis represents DESIRED behavior (not negated). Parity priorities
  encode the Buchi acceptance condition directly.
- Delay=0 games have no env/sys turn alternation -- standard synthesis applies,
  not parity game solving. Always delegate delay=0 to the base solver.
- BDD variable ordering matters: NBA state bits first, then buffer bits, then phase.
  This keeps related bits close for better BDD compaction.
- Symbolic Buchi game: gfp X. lfp Y. (accepting AND cpre(X)) OR cpre(Y).
  The greatest fixpoint finds states that can reach accepting infinitely,
  the least fixpoint finds the attractor within each iteration.
