## What I Built

C140: Neural Network -- a feedforward neural network framework with backpropagation, built from scratch.

### Components

**Tensor** -- Lightweight multi-dimensional array with:
- 1D/2D support, broadcasting, matrix multiply, transpose
- Element-wise ops, sum/mean along axes, argmax, clip
- Random initialization (normal, uniform)

**Layers:**
- Dense (fully connected) with Xavier/He/LeCun initialization

**Loss Functions:**

**Optimizers:**
- SGD with optional momentum and weight decay

- RMSProp

**Learning Rate Schedulers:**
- StepLR (periodic decay)
- ExponentialLR (exponential decay)
- CosineAnnealingLR (cosine annealing)

**Training API:**
- Sequential model (add layers, forward/backward, predict)
- fit() with mini-batching, shuffling, validation, scheduling
- evaluate(), predict_classes(), accuracy()

**Data Utilities:**
- one_hot encoding, normalize, train_test_split
- Data generators: XOR, spirals, regression, circles
- Save/load weights (dict-based)
- build_model() convenience function

### Key Design Decisions

1. **Manual backprop over AD composition** -- While C128 AD was studied, I chose manual gradient computation for each layer. This is more efficient (no computation graph overhead) and gives finer control. The manual approach matches how real frameworks work.

2. **Tensor as lightweight wrapper** -- Not a full NumPy clone, just enough for NN operations (matmul, broadcast add, element-wise ops). Keeps complexity low.

3. **Inverted dropout** -- Scale activations during training by 1/(1-p) so no scaling needed at inference. Standard modern approach.

4. **Gradient checking validates correctness** -- Numerical gradients (central differences) match analytical gradients within 1e-4 tolerance.
