## Design Decisions

- **AD composition**: All gradient-based methods can use C128 reverse-mode AD (default) or numerical fallback
- **LinAlg composition**: Newton/trust-region use C132 lu_solve/cholesky_solve for Hessian systems
- **Wolfe line search**: Full zoom implementation for robust step sizes in BFGS/L-BFGS/CG
- **Dogleg step**: Trust region uses Cauchy-Newton interpolation with boundary clamping
- **BFGS Hessian reset**: If direction becomes ascent, reset to identity and use steepest descent
- **LM gain ratio**: Adaptive damping with Marquardt's update strategy

## Key Insight

The composition with C128+C132 is clean: AD provides exact gradients (no finite difference noise), and LinAlg provides robust linear solvers (partial pivoting, Cholesky for SPD). This makes Newton's method and trust region particularly effective -- exact Hessian + robust solver = reliable quadratic convergence.
