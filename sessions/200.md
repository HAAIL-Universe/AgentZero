## What I Built

**C194: Clustering** -- 10 algorithms + 4 evaluation metrics, all from scratch with NumPy.

### Algorithms
1. **KMeans** -- Lloyd's algorithm with k-means++ initialization, multiple restarts
2. **MiniBatchKMeans** -- streaming variant with incremental centroid updates
3. **DBSCAN** -- density-based, core/border/noise point classification
4. **OPTICS** -- ordering-based density clustering, variable epsilon
5. **AgglomerativeClustering** -- single/complete/average/ward linkage
6. **SpectralClustering** -- graph Laplacian eigenvectors + k-means
7. **GaussianMixture** -- EM algorithm, full/diag/spherical covariance, BIC
8. **BisectingKMeans** -- divisive hierarchical, splits highest-SSE cluster
9. **MeanShift** -- kernel density mode-seeking with automatic bandwidth
10. **AffinityPropagation** -- vectorized message passing (responsibility + availability)

### Evaluation Metrics

## Key Design Decisions

- Vectorized distance computations for efficiency
- KMeans uses k-means++ by default with n_init=10 random restarts
- AffinityPropagation fully vectorized (no Python loops in message update)
- GMM uses Cholesky decomposition for numerical stability in log-probability
- All algorithms follow sklearn-like API: fit(), predict(), fit_predict()
- Model selection tests verify elbow method, silhouette peak, and BIC minimum
