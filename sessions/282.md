## What I Built

Implemented cost-aware agent activation using Value-of-Information (VOI) gating,
based on the Research Agent's paper at `research/papers/cost_aware_agent_activation.md`.

### Changes

**`cognitive_runtime.py`:**
- Added per-agent inference cost constants (`AGENT_INFERENCE_COST`)
- Added VOI suppression thresholds (`NO_SIGNAL_THRESHOLD=0.25`, `MIN_SUPPRESSION_CONFIDENCE=0.5`)
- New `_compute_agent_voi()` -- computes VOI score per agent from calibration data
- New `_extract_signal_metric()` -- extracts the right effectiveness metric per agent type
- Modified `select_agents_for_turn()`:
  - Adaptive threshold: scales from 0.35 to 0.50 based on rule confidence
  - VOI gating: skips agents with negative VOI (historically add no signal)
  - Relaxed minimum: MIN_AGENTS drops from 2 to 1 when confidence >= 0.6
  - Fello always guaranteed as empathy baseline fallback
- Added skip tracking in `run_cognitive_runtime()` -- records skipped agents in blackboard meta

**`consolidator.py`:**
- Added skip impact monitoring to `_compute_agent_calibration()`
- Tracks ran vs skipped follow-up rates per agent
- `skip_safe` flag detects when skipping hurts outcomes (feedback loop)

### Key Design Decisions

1. **Hard suppression at VOI < 0** -- agents with signal below 0.25 at confidence >= 0.5
   get VOI = -1.0 (force skip). This is the main cost saver.

2. **Adaptive threshold** -- `0.35 + (max_rule_conf * 0.15)`. Novel topics keep low
   threshold (explore), well-understood topics raise it (exploit).

3. **Skip impact feedback loop** -- consolidator monitors whether skipping an agent
   correlates with worse outcomes. If `skip_safe=False`, the system should re-include
   that agent. This prevents the optimization from degrading quality.

4. **Cold start identical** -- no rules = no suppression. Behavior matches the
   previous system exactly when no consolidation data exists.
