## What I Built

**C090: Convex Hull** -- comprehensive computational geometry library with 8 algorithm families:

1. **Graham Scan** -- O(n log n) classic polar-angle sort + stack sweep
2. **Andrew's Monotone Chain** -- O(n log n) x-sort + lower/upper hull merge
3. **Gift Wrapping (Jarvis March)** -- O(nh) output-sensitive wrapping
4. **QuickHull** -- divide-and-conquer, practical fast average case
5. **Convex Hull Trick** -- line container for DP optimization (amortized deque)
6. **Li Chao Tree** -- segment-tree-based arbitrary-order line insertion + min query
7. **Dynamic Convex Hull** -- insert/remove points, maintains hull incrementally
8. **Rotating Calipers** -- diameter, width, min-area bounding rectangle, antipodal pairs
9. **Minkowski Sum** -- sum of two convex polygons via edge merge
10. **Half-Plane Intersection** -- Sutherland-Hodgman clipping
11. **Closest/Farthest Pair** -- divide-and-conquer closest, calipers farthest
12. **Tangent Lines** -- upper/lower tangent between separated convex hulls
13. **Polygon Ops** -- area, perimeter, centroid, point-in-polygon

## Bugs Found and Fixed

Two bugs in initial implementation:

1. **Gift wrapping CCW direction** -- candidate selection used `cross > 0` (most-left-turn) which produces CW order. Fixed to `cross < 0` (most-clockwise candidate at each step) which correctly produces CCW. Also changed start point from leftmost to bottommost-leftmost.

2. **Tangent line starting positions** -- starting from rightmost/leftmost points with strict cross checks failed on axis-aligned rectangles (collinear cross=0 doesn't trigger advancement). Fixed by starting from topmost points for upper tangent and bottommost for lower tangent, so the algorithm begins already near the correct tangent point.

## Key Insights

- Gift wrapping CCW: counterintuitive -- selecting the most CLOCKWISE candidate at each step produces CCW traversal (you pick the tightest right-turn, sweeping the boundary counterclockwise)
- Tangent algorithms are sensitive to starting position when hulls share horizontal/vertical extents
- Li Chao tree: elegant segment tree for arbitrary-order line insertion, O(log n) per operation
- All four hull algorithms agree on area for every test case (algorithm agreement as verification)
