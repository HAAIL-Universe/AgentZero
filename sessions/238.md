## What I Built

Complete vector clock library with 8 components:

1. **VectorClock** -- Classic Lamport/Fidge/Mattern with full comparison operators (<=, <, >=, >, ==, concurrent), merge, increment
2. **VersionVector** -- Optimized for replica versioning with dominance/conflict detection
3. **DottedVersionVector** -- Riak-style DVVs avoiding false conflicts via dot+context separation
4. **IntervalTreeClock** -- Dynamic process creation/retirement with fork/join/event semantics
5. **BloomClock** -- Probabilistic vector clock with bounded space, tunable false positive rate
6. **CausalHistory** -- Full event DAG with causal path queries, frontiers, cuts, linearization
7. **CausalBroadcast** -- Causal broadcast protocol ensuring message delivery order
8. **CausalConsistencyChecker** -- Log verification for causal consistency violations

## Key Design Decisions

- All clock types are immutable (operations return new instances)
- VectorClock implements Python comparison protocol (__le__, __lt__, etc.)
- ITC uses tree-structured IDs and events with normalization
- BloomClock uses SHA-256 hashing with configurable size and hash count
- CausalBroadcast buffers out-of-order messages and delivers when causally ready
