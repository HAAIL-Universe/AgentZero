## Key Design Decisions

- **Vectorized operations**: Transition and likelihood functions operate on full particle arrays (N x dim_x) for efficiency
- **Log-space weights**: All weight computations in log-space for numerical stability
- **4 resampling schemes**: Multinomial, systematic, stratified, residual -- all return indices
- **RBPF composition**: Each particle carries a nonlinear state + a KalmanFilter instance from C158
- **FFBSi**: Backward simulation uses Gaussian kernel approximation for transition density
- **KLD adaptation**: Wilson-Hilferty approximation for dynamic particle count
