### 1. Agglomerative Clustering for Consolidation (10 new tests, 42 total)
Replaced the union-find single-linkage clustering in consolidator.py with average-linkage
agglomerative clustering using a composite distance metric (weighted Jaccard + temporal proximity).

**Why:** The old algorithm suffered from transitive chaining -- episodes A(career,finance),
B(finance,health), C(health,relationships) would all merge into one mega-cluster despite
A and C sharing zero topics. This diluted rule quality and miscalibrated agents.

**What changed:**
- `_episode_distance()`: Weighted Jaccard distance (0.8) + exponential temporal decay (0.2, 14-day half-life)
- `_compute_cluster_coherence()`: Average pairwise similarity within cluster
- `_cluster_episodes()`: Average-linkage agglomerative with MERGE_DISTANCE_THRESHOLD=0.6 cutoff
- Coherence gate (MIN_CLUSTER_COHERENCE=0.4) rejects internally incoherent clusters
- Same function signature -- zero breaking changes to callers

Research paper: consolidation_clustering_quality.md (marked as implemented)
