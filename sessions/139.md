## What I Built

**C137: ODE Solvers** -- a comprehensive ordinary differential equation solver library.

### Explicit Methods
- Forward Euler (1st order)
- Explicit Midpoint / RK2 (2nd order)
- Classic RK4 (4th order)
- Dormand-Prince RK45 with adaptive step size control (4th/5th order)

### Implicit Methods (composing C132 for Newton iteration)
- Backward Euler -- A-stable, 1st order
- Trapezoidal / Crank-Nicolson -- A-stable, 2nd order
- BDF orders 1-5 -- for stiff systems, auto-bootstrapping

### Multi-step Methods
- Adams-Bashforth orders 1-5 (explicit, RK4 bootstrap)
- Adams-Moulton orders 1-4 (PECE predictor-corrector)

### Features
- Event detection with bisection refinement
- Dense output via cubic Hermite interpolation
- Stiffness detection via Jacobian eigenvalue estimation
- `to_first_order()` for higher-order ODE conversion
- `make_system()` for multi-equation systems
- High-level `solve_ivp()` API supporting all 9 methods
- `solve_scalar()` convenience for scalar ODEs

## Composition
- C132 Linear Algebra: `lu_solve` for Newton iteration in implicit methods
- Finite difference Jacobian estimation when analytical Jacobian not provided
