## What I Built

**C149: Autoencoder / VAE** -- composing C140 (Neural Networks)

Six model variants:
1. **Autoencoder** -- standard encoder-decoder with reconstruction loss
2. **Denoising Autoencoder** -- learns to reconstruct from corrupted input
3. **Sparse Autoencoder** -- KL-divergence sparsity penalty on latent activations
4. **VAE** -- variational autoencoder with reparameterization trick
5. **Conditional VAE** -- condition on labels for controlled generation
6. **Beta-VAE** -- tunable KL weight for disentangled representations

Key components:
- Encoder/Decoder building blocks (composable, arbitrary depth)
- VAEEncoder with dual mu/log_var heads
- Reparameterization trick (backprop through stochastic sampling)
- KL divergence computation and gradients
- AnomalyDetector (threshold-based on reconstruction error)
- Latent space utilities (interpolation, statistics)
- Training utilities (fit_autoencoder, fit_vae with batching)
- Data generators (sine waves, Gaussian mixtures, binary patterns)

## Design Decisions

- VAE backward pass: full reparameterization gradient chain (dL/dmu via dL/dz + beta*dKL/dmu, dL/dlogvar via chain rule through exp(0.5*logvar)*eps)
- Conditional VAE: concatenate one-hot labels to both encoder input and decoder input
- Sparse AE: average absolute activation as sparsity measure, KL(rho || rho_hat) penalty
- All models support both MSE and BCE reconstruction loss
- Anomaly detection: fit threshold on normal data at configurable percentile
