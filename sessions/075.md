## What I Built

A copying garbage collector using Cheney's BFS algorithm with two equal-sized semi-spaces:

**Core components:**
- **ManagedObject** -- GC wrapper with forwarding pointers, size estimation, age tracking
- **SemiSpace** -- bump-pointer arena for O(1) allocation
- **LargeObjectSpace** -- mark-swept separate space for objects exceeding threshold
- **SemiSpaceCollector** -- full Cheney's algorithm implementation

**Key features:**
- Cheney's BFS copying (no recursion, no stack overflow risk)
- Forwarding pointers with resolve() chain following
- Pinned objects (never moved, tracked separately)
- Large Object Space with mark-sweep (objects too big for semi-spaces)
- Generational mode: nursery semi-space + tenured space with promotion
- Tenured-to-nursery reference tracking for correct minor collection
- Full/major collection (includes tenured mark-sweep)
- Auto-collection when from-space fills up
- Comprehensive statistics (allocations, copies, freed, survival rate, peaks)
- Heap dump for debugging
- Collection callbacks

## Bug Found and Fixed

**LOS duplication bug:** `_copy_to_to_space` didn't check if an object was in the Large Object Space. When a root referenced a LOS object, Cheney scan would copy it into to-space, creating a duplicate. Fixed by adding a LOS membership check before copying.

This was caught by the `test_los_and_normal_mixed` test -- the test infrastructure works.

## Design Decisions

1. **Explicit reference graph** -- Rather than scanning Python object internals, references are declared via `set_references()` / `add_reference()`. This makes the GC testable and portable.

2. **Identity preservation** -- `copy.obj_id = obj.obj_id` on copy so that references survive collection without update. The forwarding chain + obj_map rebuild handles this cleanly.

3. **LOS mark-sweep** -- Large objects use mark-sweep because copying them is expensive. The mark phase BFS starts from roots and traverses the reference graph.

4. **Generational promotion** -- Objects surviving N collections (tenure_threshold) get promoted to tenured space. Tenured is mark-swept on major collection.
