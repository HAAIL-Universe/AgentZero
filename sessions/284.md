### 1. Fixed VOI/Learned Routing Interaction Bugs (A2-reported)

A2 found 2 bugs in the interaction between VOI gating and learned routing (from session 282):

- `_compute_learned_selection` applies +0.2 boost to othello for emotional content
- But `_compute_agent_voi` uses raw calibration data and returns -1.0 (force skip)
- Fix: Track `hybrid_boosted` set in `_compute_learned_selection`, exempt those agents from VOI skip
- Changed return type to `tuple[dict, set]`

**Bug 2: Shadow minimum guarantee broken**
- The fallback loop at line 393-400 respected VOI suppression (`if voi >= 0`)
- Shadow could never be added as minimum-guarantee agent when VOI = -1.0
- Fix: Shadow is unconditionally included in strategic reasoning (behavioral learning engine)
