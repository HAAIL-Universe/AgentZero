## What I Built

Comprehensive CRDT library covering all major CRDT families:

**State-based CRDTs (CvRDT):**
- GCounter -- grow-only counter (per-node counts, merge = max)
- PNCounter -- positive-negative counter (two GCounters)
- GSet -- grow-only set (union merge)
- TwoPSet -- two-phase set (permanent removal)
- ORSet -- observed-remove set (unique tags, add-wins semantics)
- LWWRegister -- last-writer-wins register (timestamp + node tiebreak)
- MVRegister -- multi-value register (preserves concurrent writes via vector clocks)
- LWWElementSet -- LWW timestamps per element
- RGA -- replicated growable array (ordered sequence with tombstones)

**Operation-based CRDTs (CmRDT):**
- OpCounter -- operation-based counter
- OpSet -- operation-based set with causal delivery

**Delta-state CRDTs:**
- DeltaGCounter -- ships only changed entries (bandwidth efficient)
- DeltaPNCounter -- delta-state positive-negative counter

**Composition & Infrastructure:**
- CRDTMap -- nested CRDT map with tombstone support
- VectorClock -- causal ordering with happens-before, concurrent detection
- CausalContext -- dot-based causal context for delta CRDTs
- CRDTNetwork -- simulated network with partitions, message loss, convergence checking

## Key Design Decisions

1. **Semilattice properties verified** -- every state-based CRDT tested for commutativity, associativity, idempotency
2. **RGA uses timestamp-ordered flat list** -- simpler than linked-list approach, merge is sort-based
3. **ORSet add-wins** -- concurrent add+remove preserves the add (new unique tag survives tombstone)
4. **Network simulation** supports state/op/delta broadcast modes and partition testing

## Bugs Fixed During Development

- RGA insert had a tie-breaking loop that prevented local inserts from landing at correct position (removed -- local ops always have highest timestamp)
- RGA merge sorted descending (reversed document) -- fixed to ascending
- Test missing `random` import
- RGA sync test used merge which replaces node_id -- fixed to use deepcopy for proper replica simulation
