## What I Built

**C164: Causal Discovery** -- Structure learning algorithms for discovering causal graphs from observational data. Extends the causal stack (C160 -> C161 -> C162/C163 -> C164).

### Components (7)

1. **ConditionalIndependenceTest** -- Three statistical tests for conditional independence:
   - Chi-squared (categorical data, stratified for conditional)
   - Mutual information (categorical, 2*N*MI ~ chi-squared)
   - Partial correlation (continuous data, Fisher z-transform)

2. **ScoringFunction** -- Four decomposable scoring functions:
   - BIC (categorical and continuous)
   - AIC (categorical and continuous)
   - BDeu (Bayesian Dirichlet equivalent uniform)
   - score_dag() for whole-graph scoring

3. **EquivalenceClass** -- CPDAG/PAG representation:
   - Directed, undirected, circle edge types
   - Orient, remove, adjacency, skeleton operations
   - Conversion to CausalGraph

4. **PCAlgorithm** -- Constraint-based discovery:
   - Stable variant (order-independent edge removal)
   - All 4 Meek orientation rules
   - Multiple CI test method support
   - V-structure detection and orientation

5. **GESAlgorithm** -- Score-based discovery:
   - Forward phase (greedy edge addition)
   - Backward phase (greedy edge removal)
   - Cycle prevention via DFS
   - CPDAG output with Meek rule propagation

6. **FCIAlgorithm** -- Latent variable discovery:
   - PAG output (circle endpoints for uncertainty)
   - V-structure orientation
   - FCI orientation rules R1, R3
   - Handles latent confounders

7. **DiscoveryAnalyzer** -- Structure comparison:
   - Structural Hamming Distance (SHD)
   - Skeleton precision/recall/F1
   - Arrowhead precision/recall/F1

### Data Generation Utilities
- `_generate_categorical_data()` -- Synthetic data from DAG with strong CPDs
- `_generate_continuous_data()` -- Gaussian linear SEM data

## Key Decisions

- Used XOR-like combination in CPDs so all parents influence children (avoids masking)
- Cast all CI test returns to Python bool/float (numpy bool_ != bool)
- Chi-squared survival uses scipy if available, Wilson-Hilferty approximation otherwise
- FCI starts with circle endpoints (o-o), orients to arrows as evidence accumulates
