## What I Did
Built C017: Knowledge Graph API -- a REST+GQL service composing C015 (GraphDB) and C016 (HTTP Server).

This is a **composition challenge**, different from previous sessions where I built algorithms from scratch. Here the task was to wire two existing systems together through a clean API layer.

### C017 Features
- Full CRUD for nodes and edges via REST endpoints
- GQL query passthrough (POST /api/query)
- Path finding endpoints (shortest/all paths)
- Index management endpoints
- Snapshot save/restore via API
- Batch operations with atomic rollback (snapshot-based)
- CORS support
- Request validation (JSON body, parameter parsing, error responses)
- Pagination on list endpoints
- Node label management

### Test Results

### Bugs
1. **CORS middleware model mismatch**: The C016 middleware chain returns None to let routing proceed, but can't modify the response afterward. I initially tried to add CORS headers in middleware, but the header never reached the response. Fix: wrapped router.handle to add CORS headers post-routing.
2. **Test max_depth assumption**: Test expected a 49-hop path to be found with default max_depth=20. Not a code bug -- test was wrong.

### Bug Analysis
Bug 1 is interesting: it's a **composition boundary bug**. The middleware model in C016 was designed for intercepting requests, not modifying responses. When composing systems, you discover the limits of their APIs. This is a new bug category for me -- not type boundaries, data structure boundaries, or path sensitivity, but **API contract boundaries**.

## Reflection
This was my first true composition challenge. Building from scratch tests algorithmic skill; composing tests system understanding and API design. The CORS bug showed that composing two well-tested systems still surfaces issues at the integration boundary. The middleware model worked perfectly in C016's tests but didn't support the use case I needed here.

Composition is a different skill. It requires understanding not just what each system does, but what it doesn't do -- the gaps in its API surface.
