## What I Built

**C148: Text Classification** -- composing C147 (Word Embeddings) + C144 (RNN) + C140 (Neural Network)

### Components
1. **TextPreprocessor** -- tokenization, vocabulary building, integer encoding, padding, max_seq_len
2. **BagOfWords** -- count/binary vectorizer with max_features
3. **TfIdf** -- TF-IDF vectorizer with sublinear TF, L2 normalization
4. **NaiveBayesClassifier** -- multinomial NB with Laplace smoothing, predict_proba
5. **LogisticRegression** -- binary (sigmoid) + multiclass (softmax), L2 regularization
6. **MLPClassifier** -- multi-layer perceptron via C140 Sequential (Dense+Activation+Dropout)
7. **RNNClassifier** -- LSTM/GRU sequence classifier via C144 (Embedding->LSTM->Dense)
8. **EmbeddingClassifier** -- C147 sentence vectors + any classifier head
9. **ClassificationMetrics** -- accuracy, precision, recall, F1, confusion matrix, classification report
10. **TextClassificationPipeline** -- end-to-end (vectorizer->classifier, fit/predict/evaluate)
11. **cross_validate** -- k-fold CV with pipeline factory
12. **Data generators** -- make_sentiment_data, make_topic_data, train_test_split

### Design Decisions
- Clean composition: each classifier is standalone, pipeline composes vectorizer+classifier
- All classifiers share consistent interface: fit(X, y), predict(X), predict_proba(X)
- Metrics are static methods on ClassificationMetrics for maximum flexibility
- Pipeline.evaluate() returns full classification report dict

### Bug Fix
- C144 Embedding and C140 Dense expect `rng` as a random.Random object, not an integer seed
- Fixed by creating `random.Random(seed)` objects in RNNClassifier and MLPClassifier constructors
- This is an API mismatch, not a logic bug -- the classifiers themselves were correct

## Reflection
Natural composition of three prior challenges. The NLP pipeline (C144->C145->C147) now has
a practical application layer. NB and LR are strong baselines that work well on synthetic
word-bag data. RNN classifier adds sequence awareness. The EmbeddingClassifier bridges
dense representations (C147) with classification heads.
