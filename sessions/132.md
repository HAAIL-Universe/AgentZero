### Key Design Decisions

1. **VectorOps composition** -- All solvers use V.add/V.sub/V.scale for state arithmetic. Clean, no numpy needed.
2. **Numerical Jacobians** -- Implicit methods compute Jacobians via finite differences, then use V.solve_linear (Gaussian elimination) for Newton steps.
3. **Dormand-Prince Butcher tableau** -- Full 7-stage embedded pair with PI step controller (0.9 * err^(-0.2) with [0.2, 5.0] clamping).
4. **Event detection** -- bisection on zero crossings between accepted steps.
5. **Sensitivity via augmented system** -- Forward sensitivity propagates dim*dim extra state variables for the variational equation.
6. **Symplectic integrator** -- Stormer-Verlet preserves Hamiltonian structure; drift < 1e-4 over 20pi for harmonic oscillator.
