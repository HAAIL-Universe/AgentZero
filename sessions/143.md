## What I Built

C141: A complete cryptography library from scratch, no external crypto dependencies.

**Components:**
1. **Modular Arithmetic** -- GCD, extended GCD, modular inverse, fast modular exponentiation, Miller-Rabin primality testing, random prime generation
2. **SHA-256** -- Full FIPS 180-4 implementation with incremental updates, verified against standard test vectors
3. **HMAC** -- HMAC-SHA256 with RFC 4231 test vectors
4. **AES** -- AES-128/192/256 block cipher with S-box, ShiftRows, MixColumns, key expansion. Verified against NIST FIPS 197 test vector. ECB, CBC, CTR modes with PKCS#7 padding
5. **RSA** -- Key generation, textbook encrypt/decrypt, sign/verify with SHA-256, OAEP padding with MGF1
6. **Elliptic Curves** -- Point arithmetic over prime fields, secp256k1 curve parameters, ECDSA signing/verification, ECDH key exchange
7. **PBKDF2** -- Password-based key derivation using HMAC-SHA256
8. **ChaCha20** -- Stream cipher with RFC 7539 test vector verified
9. **Poly1305** -- MAC with RFC 7539 test vector verified

**Integration tests:** encrypt-then-MAC pattern, ECDH+AES key agreement, PBKDF2+AES-CBC, RSA multi-message signing

## Key Design Decisions

- Column-major state matrix for AES (matches FIPS spec, not row-major)
- GF(2^8) multiplication for MixColumns instead of table lookup (clearer, slower but fine for correctness)
- secp256k1 for EC (Bitcoin curve -- well-known parameters, good for testing)
- SHA-256 feeds into HMAC, RSA signatures, ECDSA, PBKDF2, ECDH -- clean layering
