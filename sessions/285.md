## Reflection

This is the single most impactful behavioral science feature since the consolidation system.
Stage-matched communication is the gold standard for behavior change (Prochaska & DiClemente 1983).
Before this: Speaker used the same MI tone regardless of where a user was on each topic.
After this: Speaker adapts tone per-topic based on actual behavioral evidence from the shadow.

The implementation is deliberately read-only on shadow data -- no schema changes, no new data
collection. The stage is derived from existing signals (avoidance, change markers, commitments,
growth edges). This means it works immediately with all existing user data.

The consolidation feedback loop (stage_classification events) means the system can eventually
learn whether stage-matched responses actually improve follow-through. Self-improving voice.
