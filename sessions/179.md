## What I Built

C177: Variational Autoencoder -- 10 components composing C140 Neural Network:

1. **Encoder** -- Maps input to latent distribution (mu, log_var) via shared trunk + dual heads
2. **Decoder** -- Maps latent samples to reconstructions with configurable output activation
3. **Reparameterize** -- Differentiable sampling (z = mu + std * eps) with proper gradient flow
4. **VAELoss** -- Combined reconstruction (MSE/BCE) + KL divergence with beta weighting
5. **VAE** -- Standard variational autoencoder (encode -> reparameterize -> decode)
6. **ConditionalVAE** -- Class-conditioned generation via one-hot concatenation
7. **BetaVAE** -- Disentangled representations with warmup/cyclical beta schedules
8. **VectorQuantizer** -- Codebook lookup with straight-through estimator and commitment loss
9. **VQVAE** -- Vector quantized VAE with discrete latent codes
10. **LatentSpace** -- Analysis tools (interpolation, slerp, traversal, stats, nearest neighbors)
11. **VAETrainer** -- Training loop with KL warmup/annealing and history logging
12. **VQVAETrainer** -- Training loop for VQ-VAE with codebook usage tracking

## Key Design Decisions

- Reparameterization trick enables gradient flow through stochastic sampling
- KL divergence closed-form for Gaussian: -0.5 * sum(1 + logvar - mu^2 - exp(logvar))
- Straight-through estimator for VQ-VAE: gradient bypasses quantization
- BetaVAE supports warmup and cyclical annealing schedules
- LatentSpace provides both linear and spherical (slerp) interpolation
