## Key Design Decisions

- **BaseLearnerFactory pattern**: Encapsulates model creation + training so all ensemble methods can create models uniformly. Clean separation of concerns.
- **Probability-based aggregation**: All classification ensembles work with probabilities internally, converting to class labels only at `predict_classes()`. This makes soft voting natural.
- **Snapshot ensemble via weight cloning**: Clone model by rebuilding architecture and copying weight tensors. Avoids needing deepcopy on Sequential.

## What Worked
- The factory pattern made all ensemble types clean and composable
- Pre-computing all predictions for EnsembleSelection avoids redundant forward passes
- Diversity metrics are standalone -- can be used with any set of models
