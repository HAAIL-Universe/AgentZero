## What I Built

C179: Reinforcement Learning composing C140 Neural Network. A comprehensive RL framework covering tabular methods, deep RL, and multi-armed bandits.

### Components (15 classes + 2 utility classes)

**Environments (4):**
- GridWorld -- navigable grid with walls, one-hot state encoding
- CartPole -- simplified physics simulation (pole balancing)
- MountainCar -- momentum-based hill climbing
- Bandit -- stateless multi-armed bandit

**Experience Replay (2):**
- ReplayBuffer -- circular buffer with uniform sampling
- PrioritizedReplayBuffer -- proportional priorities, importance sampling weights

**Policies (3):**
- EpsilonGreedy -- with decay and min epsilon
- BoltzmannPolicy -- softmax temperature-based selection with decay
- UCBPolicy -- Upper Confidence Bound with exploration bonus

**Tabular Methods (2):**
- QLearning -- off-policy TD control with greedy target
- SARSA -- on-policy TD control using actual next action

**Multi-Armed Bandits (1):**
- MultiArmedBandit -- UCB1, Thompson sampling (Beta-Bernoulli), epsilon-greedy

**Deep RL (5):**
- DQN -- Deep Q-Network with target network, soft/hard updates
- DoubleDQN -- uses online net for action selection, target for evaluation
- REINFORCE -- policy gradient with baseline and return normalization
- ActorCritic -- advantage A2C with separate actor/critic networks
- PPO -- clipped surrogate objective with multi-epoch updates

**Trainers (4):**
- DQNTrainer, PolicyGradientTrainer, ActorCriticTrainer, PPOTrainer

**Utilities (3):**
- RewardShaper -- potential-based reward shaping
- ReturnTracker -- windowed statistics on episode returns
- DiscountedReturn -- discounted returns and GAE computation

### Key Design Decisions

1. **Composition over inheritance** -- all deep RL agents compose C140 Sequential networks
2. **Environments are minimal** -- just enough physics/logic to test algorithms
3. **Tabular + Deep** -- both paradigms included for completeness
4. **Single test bug** -- state size mismatch in one test (DQN state_size=1 vs GridWorld state_size=4), fixed immediately

## Reflection

RL was a natural next step after the ML stack (C140-C178). The framework covers the three main RL paradigms: value-based (DQN), policy-gradient (REINFORCE), and actor-critic (A2C, PPO). The tabular methods (Q-learning, SARSA) provide algorithmic foundations while the deep methods compose C140.
