## What I Built

**C099: Graph Coloring** -- a comprehensive graph coloring library covering:

- **Graph data structure**: adjacency-list undirected graph with complement, subgraph, copy
- **Vertex ordering strategies**: natural, largest-first, smallest-last, random
- **Coloring algorithms**: greedy (with any ordering), DSatur (saturation degree), Welsh-Powell
- **Exact chromatic number**: backtracking with pruning, greedy clique lower bound, DSatur upper bound
- **k-colorability testing**: backtracking decision procedure
- **Graph constructors**: complete, cycle, complete bipartite, Petersen, wheel, crown, grid
- **Special graph recognition**: bipartite, chordal (MCS-based PEO), complete, cycle
- **Edge coloring**: Vizing bounds, greedy edge coloring
- **Fractional chromatic bounds**: independence number approximation
- **Interval graph coloring**: sweep-line optimal coloring
- **Register allocation**: via interval graph coloring
- **Chromatic polynomials**: tree, cycle, complete graph formulas
- **Map coloring heuristic**: smallest-last ordering for planar graphs
- **Connected components**: BFS-based

## Key Design Decisions

- Standalone module (no composition needed) -- graph coloring is a self-contained domain
- Graph uses adjacency sets for O(1) edge lookup
- Chromatic number uses DSatur as upper bound and greedy clique as lower bound, narrowing the backtracking search
- Timeout support in exact algorithms for larger graphs
