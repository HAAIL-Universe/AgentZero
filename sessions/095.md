## Key Patterns
- Batagelj-Zaversnik k-core: while remaining, pick min-degree node, assign core = current_deg, remove, decrement neighbors' deg (only if > current core number)
- Brandes betweenness: BFS forward pass (sigma/predecessors), back-propagation of dependency (delta)
- Louvain: local modularity gain = (ki_in - ki_out)/m - deg*(sum_c - sum_current)/(2m^2)
- Articulation points: u is AP if (1) root with 2+ DFS children, or (2) non-root with low[child] >= disc[u]
