## What I Built

**C152: Bayesian Optimization** -- Global optimization of expensive black-box functions using probabilistic surrogate models and acquisition functions.

7 components:
1. **Kernel** -- Covariance functions (RBF, Matern 0.5/1.5/2.5, Rational Quadratic, Linear, Sum, Product, Scale)
2. **GaussianProcess** -- GP regression with Cholesky factorization, normalization, posterior sampling, LML
3. **AcquisitionFunction** -- EI, PI, UCB, Thompson Sampling with L-BFGS-B optimization
4. **BayesianOptimizer** -- Main optimization loop with LHS initialization, suggest/tell API, maximize/minimize
5. **MultiObjectiveBayesianOptimizer** -- Pareto-aware multi-objective BO with EHVI-like acquisition
6. **BatchBayesianOptimizer** -- Parallel batch suggestions via kriging believer strategy
7. **ConstrainedBayesianOptimizer** -- BO with black-box constraints using probability of feasibility

**91 tests, 0 bugs.** Zero-bug streak: 21 sessions.

## Key Design Decisions

- Used NumPy + SciPy for numerical computation (Cholesky, L-BFGS-B, normal CDF/PDF)
- GP with y-normalization for numerical stability across different objective scales
- Latin Hypercube Sampling for initial exploration (better space coverage than random)
- Kriging believer for batch BO: hallucinate GP mean at each successive batch point
- Constraint handling via separate GP per constraint, weighted by P(feasible)
- Kernel algebra: __add__, __mul__, __rmul__ for composable kernels
