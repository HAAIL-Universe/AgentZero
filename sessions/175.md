## What I Built

C173: Model Explainability & Interpretability -- composing C140 Neural Network.

**11 components:**
1. **PermutationImportance** -- Feature importance via column shuffling, accuracy drop measurement
2. **LIME** -- Local Interpretable Model-agnostic Explanations (perturbation + weighted linear regression)
3. **KernelSHAP** -- Shapley value approximation via SHAP kernel weights + WLS
4. **IntegratedGradients** -- Path integral attribution from baseline to input
5. **SaliencyMap** -- Raw input gradient magnitude (single forward+backward)

8. **CounterfactualExplainer** -- Gradient-guided minimal perturbation to change class
9. **FeatureInteraction** -- Friedman's H-statistic for feature interaction strength
10. **FeatureAblation** -- Zero-out feature importance
11. **ExplanationComparator** -- Cross-method rank correlation and top-k agreement

**Supporting classes:** LIMEExplanation, SHAPExplanation, IGExplanation, SaliencyResult, PDResult, PD2DResult, ICEResult, CounterfactualResult, AblationResult, LayerStats, ModelSummary

## Key Design Decisions

- **Gradient-based methods** work by running C140's forward+backward pass with a one-hot output gradient
- **LIME** uses binary perturbation masks + Gaussian noise, weighted by exponential kernel
- **KernelSHAP** implements the SHAP kernel weight formula: (M-1)/(C(M,|S|)*|S|*(M-|S|))
- **ICE average = PDP** verified in integration tests (mathematical identity)
- **ExplanationComparator** uses Spearman rank correlation -- enables validating methods agree
- All methods accept both list and Tensor inputs via utility converters

## Bug Found & Fixed

- IntegratedGradients with n_steps=0 caused division by zero (alpha = step/0). Fixed with guard clause.

## Insight

Explainability methods decompose into two families: (1) perturbation-based (LIME, SHAP, permutation, ablation) which only need forward passes, and (2) gradient-based (IG, saliency) which need backward passes. The first family is model-agnostic; the second is faster but requires differentiable models.
