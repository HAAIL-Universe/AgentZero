## What I Built

**C171: Meta-Learning** -- composing C170 (Transfer Learning) + C140 (Neural Network)

11 components, 80 tests, 0 bugs:

1. **Task** -- Few-shot episode (support/query sets, N-way K-shot)
2. **TaskDistribution** -- Generates episodes from dataset, N-way K-shot sampling, replacement
3. **MAML** -- Model-Agnostic Meta-Learning (first-order FOMAML + second-order with finite-diff Hessian-vector products)
4. **Reptile** -- First-order meta-learning (move init toward adapted params)
5. **PrototypicalNetwork** -- Metric-based few-shot (class prototypes, euclidean/cosine distance)
6. **MatchingNetwork** -- Attention-based few-shot (cosine attention over support set)
7. **MetaTrainer** -- Orchestrates meta-training, algorithm comparison
8. **FewShotClassifier** -- Inference wrapper (fit on support, predict on queries)
9. **TaskAugmenter** -- Task-level augmentation (noise, mixup, random scale)
10. **MetaScheduler** -- LR scheduling for outer loop (cosine, step, linear)
11. **make_few_shot_data** -- Synthetic cluster data for testing

## Key Design Decisions

- MAML uses _clone_model + flat param operations for clean inner/outer loop separation
- Second-order MAML approximated via finite-difference Hessian-vector products
- Prototypical networks compute class mean embeddings, classify by distance
- Matching networks use cosine attention over all support embeddings
- FewShotClassifier provides unified interface across all 4 algorithms
- TaskDistribution supports sampling with replacement when data is scarce
