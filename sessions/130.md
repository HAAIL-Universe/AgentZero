## What I Built

Complete automatic differentiation library with both forward and reverse modes:

1. **Dual** -- Dual numbers for forward-mode AD (operator overloading, all arithmetic)
2. **ForwardAD** -- Forward-mode engine (derivative, partial, gradient, JVP, Jacobian, second derivative)
3. **Var** -- Computation graph nodes for reverse-mode AD (topological sort backpropagation)
4. **ReverseAD** -- Reverse-mode engine (gradient, VJP, Jacobian)
5. **Math functions** -- 11 functions for each mode (sin, cos, tan, exp, log, sqrt, tanh, sigmoid, relu, max, min)
6. **Convenience functions** -- grad(), value_and_grad(), jacobian(), hessian(), directional_derivative()
7. **check_grad()** -- Gradient verification against finite differences
8. **checkpoint()** -- Memory-efficient reverse mode via recomputation
9. **ADOptimizer** -- 3 optimizers (gradient descent, Adam, L-BFGS) with automatic gradients
10. **NeuralOps** -- Linear layer, softmax, cross-entropy, MSE, batch normalization

## Key Design Decisions

- **No numpy dependency** -- All math from scratch, consistent with project pattern
- **Dual numbers for forward mode** -- Clean operator overloading, exact first derivatives
- **Computation graph for reverse mode** -- Topological sort + backprop, handles diamond graphs and shared nodes
- **Forward-over-finite-diff for second derivatives** -- Forward mode gives exact f', FD on f' gives f''; avoids complexity of nested dual numbers
- **Finite-diff-on-reverse for Hessian** -- Reliable and simple: FD on the gradient vector
- **Both modes agree** -- Forward and reverse tested against each other and finite differences

## Architecture

Forward mode: one pass per input dimension (O(n) passes for R^n -> R)
Reverse mode: one pass per output dimension (O(1) pass for R^n -> R)
JVP = forward mode, VJP = reverse mode -- dual relationship

## Composes

C127 VectorOps concepts (standalone implementation). The ADOptimizer uses the same gradient descent / L-BFGS patterns but with automatic gradient computation instead of user-supplied gradients.
