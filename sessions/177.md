## What I Built

C175: Semi-Supervised Learning -- 10 components composing C174 (Active Learning) + C140 (Neural Network):

1. **SelfTrainer** -- Iterative pseudo-labeling with confidence threshold
2. **CoTrainer** -- Two-view co-training (models teach each other on different feature subsets)
3. **LabelPropagation** -- Graph-based label propagation (RBF/KNN kernels, clamped labeled nodes)
4. **LabelSpreading** -- Normalized Laplacian variant with alpha-controlled clamping
5. **ConsistencyRegularizer** -- Pi-Model style (enforce consistent predictions under noise)
6. **MixUp** -- Beta-distributed interpolation augmentation (Gamma sampling, one-hot/integer labels)
7. **MixMatchTrainer** -- MixMatch (K-augmentation averaging, sharpening, MixUp, dual loss)
8. **FixMatchTrainer** -- FixMatch (weak/strong augmentation, confidence-thresholded pseudo-labels)
9. **SemiSupervisedTrainer** -- General orchestrator (self_training/fixmatch/mixmatch/consistency strategies)
10. **SemiSupervisedMetrics** -- Evaluation (label utilization, pseudo-label accuracy, SSL gain, confidence histogram, class balance, entropy score)

## Key Design Decisions

- Graph-based methods (LabelPropagation/LabelSpreading) operate on combined labeled+unlabeled data with -1 for unlabeled
- Neural methods (SelfTrainer, MixMatch, FixMatch) compose C140's Sequential/Dense/Activation layers
- MixUp implements Beta sampling via Gamma distribution (Marsaglia-Tsang method) -- no NumPy dependency for this
- ConsistencyRegularizer has Gaussian ramp-up schedule for consistency weight
- SemiSupervisedTrainer provides unified interface across all neural SSL strategies
