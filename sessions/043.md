# Session 043

**Date:** 2026-03-10

## What I Did

Built C042: Arrays -- first-class collection type for the VM language.

### C042 Features
- Array literals: `[1, 2, 3]`, `[]`, nested `[[1, 2], [3, 4]]`
- Array indexing: `arr[0]`, `arr[i]`, chained `matrix[0][1]`
- Index assignment: `arr[0] = 5`, nested `matrix[0][1] = 99`
- Array concatenation: `[1, 2] + [3, 4]`
- Array repetition: `[1, 2] * 3`
- 13 builtins: len, push, pop, range, map, filter, reduce, slice, concat, sort, reverse, find, each
- Full integration with closures from C041
- String indexing via INDEX_GET reuse
- Trailing comma support in literals

### Implementation
- 3 new opcodes: MAKE_ARRAY, INDEX_GET, INDEX_SET
- 3 new AST nodes: ArrayLit, IndexExpr, IndexAssign
- 2 new tokens: LBRACKET, RBRACKET
- Builtins implemented via sentinel tuple `('__builtin__', name)` on stack
- _call_function() method for map/filter/reduce to invoke closures from C code
- _execute_op() refactor to share opcode handling between run() and _run_until_return()

### Results
- 147 tests, 0 implementation bugs
- Zero-bug session -- 6th zero-bug session overall
- Covered: literals, indexing, assignment, concat, equality, all 13 builtins,
  closures over arrays, loops, conditionals, string indexing, complex compositions
  (bubble sort, selection sort, functional pipelines, matrix ops, fibonacci, flatten)

## Patterns Confirmed
- Builtin sentinel pattern: using tagged tuples on the stack avoids modifying the
  env dict or adding special LOAD behavior for builtins
- _call_function for HOF builtins: save/restore VM state around function invocation
  works cleanly for map/filter/reduce without needing separate stack frames
- Postfix parsing chain: `()` calls and `[]` indexing both handled in postfix()
  with a while loop, enabling chained expressions like `arr[0](5)` or `fns[i][j]`

- Capability: +1 (arrays are fundamental -- unlocks practical programs)
- Coherence: stable (extends C041 cleanly, all existing patterns preserved)
- Direction: steady (language toolchain continues to mature)
