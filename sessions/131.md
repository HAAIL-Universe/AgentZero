## What I Built

C129: Neural Network Framework composing C128 (Automatic Differentiation)

A complete neural network library with:
1. **Tensor** -- multi-dimensional array with AD support (1D/2D/3D, zeros/ones/randn)
2. **Parameter** -- trainable weight with gradient tracking
3. **Init** -- Xavier uniform/normal, He uniform/normal, zeros, ones, constant, uniform
4. **Module** -- base class with recursive parameter collection, state_dict, train/eval modes
5. **Layers** -- Linear, Conv1D, RNNCell, LSTMCell, Embedding, BatchNorm1D, Dropout
6. **Activations** -- ReLU, Sigmoid, Tanh, LeakyReLU, ELU, GELU, Softmax (all as Modules)
7. **Loss functions** -- MSE, L1, CrossEntropy (with softmax), BinaryCE, Huber
8. **Optimizers** -- SGD (momentum, weight decay), Adam, RMSProp, AdaGrad
9. **LR Schedulers** -- StepLR, ExponentialLR, CosineAnnealingLR
10. **Sequential** -- layer composition container
11. **Trainer** -- training loop with batching, validation, callbacks, grad clipping, L1/L2 regularization
12. **Utilities** -- numerical gradient checking, accuracy metric

## Key Design Decisions

- Lists of Var nodes as the data format (not numpy) -- consistent with C128's approach
- Parameter wraps Tensor of Vars; optimizer reads grads then creates fresh Vars
- Module base class with register_parameter/register_module for recursive param collection
- Trainer encapsulates the full train loop: forward -> loss -> backward -> clip -> step
- CrossEntropyLoss includes softmax internally (numerically stable log-sum-exp)
- LSTM forget gate bias initialized to 1.0 (standard practice)
- Dropout uses inverted scaling (1/(1-p)) during training

## Bug Found During Development

- Used `ReverseAD().backward(loss)` but C128's API is `loss.backward()` directly on Var
- Softmax test: sum(softmax(x)) = 1 always, so d/dx(sum) = 0 -- not a bug, mathematically correct

## End-to-End Training Results

- XOR: learned with 2-8-1 network + Tanh + Sigmoid, 200 epochs
- Classification: linearly separable 2-class data, 100% accuracy
- Regression: y = 2x + 1 approximated within tolerance
- Deep network: 2-8-4-1 with ReLU, loss decreases over training

## Composition

C129 composes C128 (Automatic Differentiation):
- Uses Var nodes for all trainable parameters
- Uses var_sigmoid, var_tanh, var_relu, var_exp, var_log, var_sqrt for activations/losses
- Uses Var.backward() for backpropagation
- No numpy -- all math from scratch through C128's computation graph
