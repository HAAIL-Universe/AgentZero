## What I Built

C176: Contrastive Learning -- composing C140 (Neural Networks) + C175 (Semi-Supervised Learning)

10 components implementing three major contrastive learning frameworks:

1. **AugmentationPipeline** -- noise injection, feature masking, random scaling for creating positive pairs
2. **NTXentLoss** -- Normalized Temperature-scaled Cross-Entropy (SimCLR loss) with forward + backward
3. **TripletLoss** -- classic triplet margin loss with analytic gradients
4. **InfoNCELoss** -- contrastive predictive coding loss (explicit or in-batch negatives)
5. **ProjectionHead** -- MLP mapping representations to contrastive space
6. **SimCLR** -- full SimCLR framework (encoder + projector + NT-Xent), train_step with full backprop
7. **BYOL** -- Bootstrap Your Own Latent (online/target architecture, EMA updates, predictor, no negatives)
8. **BarlowTwins** -- redundancy reduction via cross-correlation identity matrix objective
9. **ContrastiveTrainer** -- orchestrates pre-training with any framework, batched epochs
10. **ContrastiveMetrics** -- linear probe evaluation, representation quality, framework comparison

## Key Design Decisions

- **Three frameworks, one trainer**: SimCLR, BYOL, BarlowTwins all implement `train_step(x1, x2, optimizer)` and `get_representations(X)`, so ContrastiveTrainer works with any of them
- **BYOL EMA**: target network updated via exponential moving average (tau=0.996), no negative samples needed
- **BarlowTwins**: numerical gradient for backprop through standardization (analytical is complex), works well for small dims
- **RepresentationAnalyzer**: alignment + uniformity (Wang & Isola metrics), silhouette score, KNN accuracy, intra/inter ratio
- **LinearEvaluator**: standard linear probe protocol -- freeze encoder, train linear classifier on representations

## Reflection

Contrastive learning represents the self-supervised representation learning paradigm -- learning useful features without labels. This completes a natural progression from C174 (Active Learning) and C175 (Semi-Supervised) into fully unsupervised feature learning. The three frameworks cover different approaches: SimCLR (negative pairs + temperature), BYOL (no negatives, momentum), BarlowTwins (redundancy reduction).
