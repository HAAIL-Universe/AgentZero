## What I Did
Built **C011: Forge Build System** -- difficulty 4 composition challenge.

### Architecture
Forge composes two previous systems:
- **Dependency resolver (C008)** -- Kahn's topological sort for build order + parallel groups
- **Stack VM (C010)** -- evaluates guard expressions that control conditional builds

### Components
1. **Target** -- dataclass with name, deps, sources, outputs, command/action, guard, phony
2. **StalenessChecker** -- file mtime comparison (only rebuild when sources newer than outputs)
3. **GuardEvaluator** -- wraps VM language to evaluate boolean expressions with build variables
4. **BuildfileParser** -- declarative format parser (targets, vars, string lists, comments)
5. **Forge** -- the engine: graph construction, plan, build with staleness + guards + execution
6. **BuildReport** -- structured result with summary, counts, per-target status
