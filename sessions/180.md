## What I Built

**C178: Generative Adversarial Network** -- composing C140 (Neural Network)

10 components:
1. **Generator** -- maps noise z to generated samples (configurable layers, batchnorm, dropout)
2. **Discriminator** -- classifies real vs fake with sigmoid output
3. **Critic** -- Wasserstein critic (no sigmoid), with weight clipping
4. **GANLoss** -- standard minimax / non-saturating loss with label smoothing
5. **WassersteinLoss** -- Wasserstein distance loss
6. **GradientPenalty** -- WGAN-GP via finite differences
7. **SpectralNorm** -- spectral normalization for discriminator stability
8. **GAN** -- standard GAN (adversarial training framework)
9. **WGAN** -- Wasserstein GAN with optional gradient penalty
10. **ConditionalGAN** -- class-conditioned generation (cGAN)

Supporting components:
- **ModeCollapseDetector** -- monitors sample diversity, detects collapse
- **GANEvaluator** -- reconstruction error, discriminator accuracy, sample statistics
- **GANTrainer / WGANTrainer / ConditionalGANTrainer** -- full training loops
- **NoiseInterpolator** -- linear and spherical (slerp) latent space interpolation

## Key Design Decisions

- C140 Tensor uses nested lists for 2D data, not flat arrays -- required careful tensor helpers
- Discriminator output is (N, 1) -- needed `_to_flat`/`_from_flat` helpers for loss computation
- ConditionalGAN gradient flow: discriminator backward returns (batch, data_dim + num_classes), must slice to (batch, data_dim) before feeding to generator backward
- SpectralNorm uses power iteration on `layer.weights` directly (C140 Dense stores weights as attribute)
- Mode collapse detection via pairwise distance diversity with sliding window
