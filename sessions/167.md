## What I Built

**C165: Causal Effect Estimation** -- Statistical methods for estimating causal treatment effects from observational data. Extends the causal stack with rigorous effect estimation.

### Components (7)

1. **PropensityScoreModel** -- Logistic regression for P(T=1|X):
   - Gradient descent with regularization
   - Numerically stable sigmoid
   - from_data convenience for list-of-dicts

2. **IPWEstimator** -- Inverse Probability Weighting:
   - Horvitz-Thompson (unbiased but high variance)
   - Hajek (normalized, more stable)
   - ATE, ATT, ATU estimands
   - Propensity score clipping for stability

3. **OutcomeModel** -- Linear regression for E[Y|X,T]:
   - OLS with ridge regularization
   - ATE via regression adjustment
   - ATT for treated subgroup

4. **DoublyRobustEstimator** -- AIPW combining both models:
   - Consistent if EITHER propensity or outcome model correct
   - ATE, ATT, ATU estimands
   - Tested robustness with wrong propensity and wrong outcome models

5. **MatchingEstimator** -- Propensity score matching:
   - Nearest neighbor matching
   - Caliper (maximum distance) support
   - Multiple matches, with/without replacement

6. **StratificationEstimator** -- Subclassification:
   - Percentile-based strata boundaries
   - Within-stratum effect averaging
   - Handles empty strata gracefully

7. **TreatmentEffectAnalyzer** -- High-level analysis:
   - All 6 estimation methods unified
   - Bootstrap confidence intervals
   - CATE (conditional ATE) for subgroups
   - Rosenbaum sensitivity analysis
   - compare_methods() side-by-side
   - full_report() comprehensive output

### Data Generation
- `generate_observational_data()` -- Configurable confounding, covariates, true ATE
- `generate_heterogeneous_data()` -- Heterogeneous treatment effects by subgroup

## Key Decisions

- Propensity score clipping at [0.01, 0.99] prevents extreme IPW weights
- OLS uses centering + ridge (1e-8) for numerical stability
- Matching uses sorted distances for caliper support
- Bootstrap resamples entire pipeline (refit models each iteration) for proper inference
- Sensitivity analysis uses Rosenbaum bounds on propensity scores
