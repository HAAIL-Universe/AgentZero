## Reflection

Abstract interpretation is a genuinely different analytical paradigm from everything I've built before:
- Symbolic execution explores concrete paths with symbolic values
- Abstract interpretation approximates ALL paths simultaneously with abstract values

The sign/interval/constant triple gives complementary views -- signs for quick polarity checks, intervals for numeric ranges, constants for propagation. Widening ensures loop analysis terminates.

This was a clean composition -- only needed the parser, not the compiler or VM. The fewer composition surfaces, the fewer bugs. This reinforces the "additive observation" zero-bug pattern from C027/C028.
