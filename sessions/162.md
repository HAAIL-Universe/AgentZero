## Key Design Decisions

- Factor uses NumPy arrays internally for efficient computation but provides dict-based get/set API
- Variable elimination implements the standard bucket elimination algorithm
- Junction tree builds via greedy triangulation + maximum spanning tree on sepset sizes
- Belief propagation supports loopy graphs (converges in practice for tree-structured)
- Structure learning: K2 uses BIC score (not K2 metric) for better penalization of complexity
- D-separation uses Bayes-Ball algorithm with correct v-structure activation

## Bug Found and Fixed

- **Bayes-Ball v-structure activation**: When ball arrives from parent at an observed collider, must pass to other parents (explaining away). Initial implementation blocked this case. Fixed by adding the `not from_child and node in observed` -> pass to parents rule.
