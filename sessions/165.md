## What I Built

C163: Structural Equation Model -- extending C161 (Causal Inference) with explicit structural equations.

10 components:
1. **StructuralEquation** -- Single equation: X_i = f(parents, noise), linear or arbitrary function
2. **LinearSEM** -- X = BX + e, path matrix, implied covariance, total/direct/indirect effects
3. **NonlinearSEM** -- Arbitrary function-based structural equations with counterfactual support
4. **SEMIdentification** -- Backdoor criterion, instrumental variable discovery, Wright's path tracing
5. **SEMIntervention** -- do(X=x) via equation replacement (not just graph surgery)
6. **SEMCounterfactual** -- Abduction-action-prediction with exact noise recovery for linear SEMs
7. **SEMEstimation** -- OLS, 2SLS, path coefficient estimation from data
8. **SEMFitMetrics** -- R-squared, residuals, implied vs observed covariance, AIC/BIC
9. **SEMSimulator** -- Monte Carlo simulation, interventional distributions, sensitivity analysis
10. **SEMAnalyzer** -- Effect decomposition, mediation analysis, full reports, unified causal queries

## Key Design Decisions

- Linear SEMs use matrix algebra: total effect = (I-B)^{-1}, implied covariance = (I-B)^{-1} Omega ((I-B)^{-1})^T
- Counterfactuals use exact noise recovery for linear case (Y_observed - Y_predicted = noise)
- Wright's path tracing validates against matrix-based total effect (cross-check)
- 2SLS correctly recovers causal effects under confounding (demonstrated in integration test)
- SEMIntervention creates new SEM with replaced equations rather than mutating

## Integration Tests Verified

- Observational vs interventional distributions differ under confounding (U->X, U->Y)
- 2SLS is less biased than OLS under confounding
- Wright's path tracing matches (I-B)^{-1} total effect
- Implied covariance converges to sample covariance with large N
- do() + sample matches direct interventional sampling
