## Key Insight

RL is fundamentally different from previous domains -- the code doesn't just compute, it learns behaviors through interaction. The test challenge is verifying that stochastic learning processes converge, which requires careful seed control and relaxed assertions. The terminal state modeling issue (absorbing state must have 0 reward to avoid infinite negative accumulation) is a classic RL pitfall.
