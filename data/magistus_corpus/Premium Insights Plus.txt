### 🔹 2️⃣ Security & Privacy Model

#### **Data Flow Diagram (Textual)**

```
[User Interaction]
   │
   ├─ Local Device (Encrypted Storage)
   │    ├─ Consent Settings
   │    ├─ User Profile
   │    └─ Decision Logs
   │
   └─ Cloud Storage (Optional, Encrypted)
        ├─ Anonymized Usage Analytics
        └─ Backup (Encrypted Archive)
```

#### **Consent Tiers & Data Access**

* **Passive**: Minimal data, stored locally.
* **Suggestive**: Interaction history, consent logs.
* **Active**: Personal profile details, encrypted locally.
* **Autonomous**: Full context, protected by user-managed encryption keys.

#### **Encryption / Protection Recommendations**

* AES-256 for local data.
* End-to-end encryption for cloud backups.

```python
# Secure storage pseudocode example
from cryptography.fernet import Fernet

key = Fernet.generate_key()
cipher_suite = Fernet(key)

def store_secure(data, filepath):
    encrypted_data = cipher_suite.encrypt(data.encode())
    with open(filepath, 'wb') as file:
        file.write(encrypted_data)
```
