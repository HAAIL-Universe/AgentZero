## Key Design Decisions
- Reuses V098's micro-function algebra and flow functions unchanged
- DemandDrivenSolver builds reverse edge index for backward traversal
- Recursion guard prevents infinite loops on cyclic dependencies (returns TOP)
- Cache invalidation supports both full reset and point-specific invalidation
- Explored points set naturally forms a demand-driven backward slice

## Bug Found
- ZERO fact not in problem.all_facts: V098's build_ide_problem only puts
  real variables in all_facts, not ZERO. But ZERO is the universal generator
  that creates new facts via ConstFunction. Missing ZERO meant zero-arg
  function returns (fn seven() { return 7; }) produced TOP instead of Const(7).
- Fix: _all_facts_with_zero() returns all_facts | {ZERO}
