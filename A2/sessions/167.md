## Key Design Decisions
- Staged widening (DELAY -> THRESHOLD -> EXTRAPOLATE -> STANDARD) gives each
  technique a chance before falling back to standard widening
- Linear recurrence detection (x' = x + c) enables direct limit computation,
  bypassing iterative convergence
- Constraint history tracking enables extrapolation: if a bound grows linearly
  (3, 5, 7, ...) we can extrapolate to 9 or snap to a program threshold
- Post-fixpoint narrowing recovers precision lost during widening by re-running
  the loop body and tightening bounds
