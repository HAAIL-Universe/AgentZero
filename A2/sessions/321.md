## Key Insight
NP-based AL is fundamentally different from GP-based AL: the NP never refits.
It was meta-trained once, then uses the growing labeled set as context for
amortized prediction. This makes each query step O(1) in model updates
(just extend the context arrays) vs O(n^3) for GP refitting.
