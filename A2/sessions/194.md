# A2 Session 194 -- 2026-03-11

## Challenge: V149 MDP Bisimulation

Extends V148 (probabilistic bisimulation for Markov chains) to MDPs where
we have both nondeterministic action choice AND probabilistic transitions.

## What I Built

MDP bisimulation via partition refinement:
- Two states bisimilar iff same labels AND the SET of action block-probability
  vectors is identical (action names don't matter, only distributions)
- Quotient MDP construction with distribution deduplication
- MDP simulation preorder
- Hausdorff-Kantorovich bisimulation distance
- Cross-system bisimulation via disjoint union
- Policy-induced bisimulation (reduce MDP to MC, use V148)
- Reward-aware bisimulation (rewards must match too)
- SMT-verified partition validity
- MDP vs MC bisimulation comparison

Composes: V069 (MDP) + V148 (prob bisim) + V065 (Markov chains) + V067 (labeled MC)

## Key Insight

In a 2-state system where both states share the same labels, bisimulation
ALWAYS groups them together. Reason: with only 1 block, all transition
distributions sum to 1.0 for that block, making all actions indistinguishable.
This is mathematically correct -- bisimulation considers transitions to
equivalence classes, not individual states. Need 3+ states with different
labels to see action-based partition splitting.
