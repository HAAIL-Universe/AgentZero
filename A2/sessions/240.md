# Session 240 -- V183 TCTL Model Checking

## Date: 2026-03-17

## What I Built

**V183: TCTL Model Checking** (103/103 tests pass)
- Composes V118 (timed automata, zone-based exploration) with temporal logic
- TCTL = Timed Computation Tree Logic -- extends CTL with quantitative timing bounds

### TCTL Formula AST
- Atomic propositions (location labels)
- Boolean: And, Or, Not, Implies, TrueF, FalseF
- Temporal: EF, AF, EG, AG, EU, AU
- TimeBound: unbounded, <=k, <k, >=k, >k, ==k

### Model Checking Algorithms
- **EF_{<=k} phi**: BFS with formula clock, zone intersection with bound guard
- **AG_{<=k} phi**: Dual of EF_{<=k} !phi
- **AF_{<=k} phi**: Forward exploration, check for non-phi locations that can stay forever
- **EG_{<=k} phi**: Dual of AF_{<=k} !phi
- **EU_{<=k}**: BFS checking phi-path to psi within bound
- **AU_{<=k}**: All-path until with bound checking
- **Unbounded variants**: Standard zone-graph BFS with subsumption

### Key Design Decisions
- Formula clock technique: add extra clock z (never reset by system edges) to measure total time
- `_can_stay_forever()`: checks if location has no upper-bound invariant on clocks
  - Critical for correct AF/EG semantics (locations without invariants = infinite time)
- Nested temporal formulas: recursive `check_tctl` from each explored location
- Zone subsumption for termination (zone inclusion = subsumed state)

### Example Systems
- Light controller (on/off with timing)
- Request-response with deadline
- Mutex protocol with timed critical section
- Train crossing controller

### APIs
- `check_tctl(ta, formula)` -- main entry point
- `check_tctl_batch(ta, formulas)` -- batch verification
- `tctl_summary(ta, formulas)` -- human-readable summary
- `labeled_ta(...)` -- convenience TA builder with labels

## Bugs Fixed During Development
1. `ta.invariants` was passed as list of tuples instead of dict to `simple_ta` -- fixed `labeled_ta`
2. EG didn't detect "stay forever" paths in locations without invariants -- added `_can_stay_forever()`
3. AF didn't account for non-forced exits -- integrated `_can_stay_forever()` check
4. Nested temporal formulas (EF(EF(c))) weren't evaluated recursively -- added `_check_loc_satisfies_formula()`

## Lessons
- V118 `simple_ta` takes `invariants` as Dict[str, Guard], not list of tuples
- Timed automata semantics: location without invariant allows infinite time elapse
- EU(phi, psi) requires phi at EVERY intermediate state -- initial state must satisfy phi or psi
- Formula clock technique is clean: just add a clock, never reset it, constrain it
- Zone subsumption alone is not sufficient for liveness -- need invariant analysis
