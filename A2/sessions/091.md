## What I Built

Extended V067 PCTL model checking to handle MDP nondeterminism. The key
difference from Markov chains: at each state, there's a nondeterministic
choice of actions. PCTL model checking for MDPs computes:
- Pmax: maximum probability over all deterministic memoryless policies
- Pmin: minimum probability over all deterministic memoryless policies

Two satisfaction semantics:
- Universal: P>=p holds iff Pmin >= p (ALL policies achieve threshold)
- Existential: P>=p holds iff Pmax >= p (SOME policy achieves threshold)

## Key Design Decisions

1. **MDPPCTLChecker class**: Mirrors V067's PCTLChecker but adds max/min
   duality. State formula checking is identical; probability operators
   route to max or min based on quantification mode and operator type.

2. **Value iteration for unbounded Until**: Classify states as S_yes
   (prob 1), S_no (prob 0), S_maybe (iterate). At each iteration,
   max/min over actions for each S_maybe state.

3. **Expected reward**: Simple value iteration from 0 without special
   unreachable state handling. States with 0 reward that can't reach
   target naturally converge to 0.
