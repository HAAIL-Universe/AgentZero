### Core: POMDP Model
Extends MDP with observation model O(o|s',a). Agent cannot see true state,
only receives noisy observations. Maintains belief state b(s) = P(s|history)
via Bayesian update.

### Belief Update
Bayesian filtering: b'(s') = eta * O(o|s',a) * sum_s T(s'|s,a) * b(s)
Normalized posterior. Handles zero-probability observations gracefully.

### 5 Solvers
1. **QMDP**: Upper bound heuristic -- solve underlying MDP, use Q-values as
   alpha vectors. Fast but assumes full observability after one step.
2. **FIB (Fast Informed Bound)**: Tighter upper bound than QMDP. Accounts for
   observation uncertainty with per-observation-weighted max over successors.
3. **Exact Value Iteration**: Alpha-vector pruning. Computes optimal value
   function. Exponential but exact. Incremental pruning for efficiency.
4. **PBVI (Point-Based Value Iteration)**: Approximate solver at finite belief
   points. Much more scalable. Belief expansion via random forward simulation.
5. **Perseus**: Randomized point-based backup. Only keeps backups that improve
   value. Most efficient for large belief sets.

### Belief-Space Analysis
- Simulation: sample-based trajectory generation under alpha-vector policy
- Evaluation: Monte Carlo expected reward estimation
- Information gain: H(b) - E_o[H(b'|o,a)] for each action
- Most informative action selection
- Belief entropy computation

### 4 Classic Problems
1. **Tiger**: 2 states, 3 actions, 2 observations. Classic information-gathering problem.
2. **Machine Maintenance**: N-condition degradation with inspect/repair decisions.
3. **Hallway Navigation**: 1D grid with noisy movement and position sensing.
4. **RockSample (small)**: Grid with hidden rock quality, check/sample actions.
