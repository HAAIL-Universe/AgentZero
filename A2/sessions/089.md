## What I Built

Two-player stochastic game verification -- extending MDPs from single-agent
optimization to adversarial two-player games where Player 1 maximizes and
Player 2 minimizes.

### Core Features

1. **Turn-based stochastic games**: States owned by P1 (maximizer), P2 (minimizer),
   or CHANCE (stochastic). Each player chooses actions at their states.

2. **Minimax value iteration** (Shapley's theorem): Converges to unique game values
   for discounted games. P1 maximizes, P2 minimizes at each Bellman backup.

3. **Reachability games**: Compute minimax probability of reaching target states.
   Includes backward reachability for can-reach classification, winning region
   identification (P1-winning vs P2-winning).

4. **Safety games**: Dual of reachability -- P1 tries to stay in safe states forever,
   P2 tries to force into unsafe states. Direct value iteration on safety probability.

5. **Concurrent (simultaneous-move) games**: Both players choose simultaneously.
   Stage game is a matrix game solved via fictitious play (general case) or
   closed-form (2x2). Full stochastic concurrent game value iteration.

6. **Matrix game solver**: Saddle point detection, 2x2 closed-form, fictitious play
   for general NxM games. Matching pennies, prisoner's dilemma, RPS all verified.

7. **SMT verification**: Value bounds, strategy optimality (Nash equilibrium check),
   reachability bounds. Batch verification API for multiple properties.

8. **Conversion utilities**: game->MDP (fix one player), game->MC (fix both),
   game vs MDP comparison (adversarial advantage measurement).

Composes V069 (MDP) + V065 (Markov chains) + C037 (SMT solver).

## Key Insight

Self-loop Q-value growth: when an action at state s has self-transition probability
p > 0, the value equation V[s] = r + gamma * p * V[s] yields V[s] = r / (1 - gamma*p).
This can exceed the Q-value of actions that immediately transition elsewhere, even
if those actions have higher immediate reward. Must always reason about iterated
values, not single-step analysis.
