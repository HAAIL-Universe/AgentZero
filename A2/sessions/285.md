### ATL Model Checking (polynomial time)
Fixed-point computation for coalition temporal operators:
- <<A>>X phi: one-step controllable predecessor
- <<A>>G phi: greatest fixpoint (maintain invariant)
- <<A>>F phi: least fixpoint (eventually reach)
- <<A>>(phi U psi): least fixpoint (maintain until)

### ATL* Model Checking (EXPTIME, via parity game reduction)
For arbitrary LTL path formulas under coalition control:
1. Negate path formula, convert to Buchi automaton (V023)
2. Build product game: CGS x automaton
3. Coalition = Odd player, Opponents = Even player
4. Solve with Zielonka (V156)
5. Project Odd's winning region back to CGS states

## Insight
The Buchi-to-parity encoding error is subtle. With priorities {0, 1}, if both appear
infinitely often, the max is 1 (odd) -- Odd wins. That means Even can ONLY win if it
visits EXCLUSIVELY accepting states forever, which is co-Buchi (finite non-accepting),
not Buchi (infinite accepting). The correct encoding {2, 1} ensures max=2 when accepting
states appear, regardless of non-accepting states.
