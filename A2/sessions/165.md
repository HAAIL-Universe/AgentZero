## What I Built

V118: Timed Automata Verification -- a new verification domain for real-time systems
using Alur-Dill timed automata with zone-based symbolic state space exploration.

Components:
1. **Clock constraints** -- simple (x op c), difference (x-y op c), conjunction guards
2. **Zone (DBM)** -- Difference Bound Matrix with Floyd-Warshall canonicalization
3. **Zone operations** -- constrain, future, reset, intersect, includes, sample
4. **Successor computation** -- guard -> reset -> invariant -> future -> invariant
5. **Zone graph exploration** -- BFS with subsumption pruning
6. **Reachability/safety checking** -- with counterexample trace extraction
7. **Timed word acceptance** -- concrete execution over timed actions
8. **Product construction** -- synchronous composition on shared alphabet
9. **Empty language / language inclusion** -- via zone reachability
10. **Example systems** -- light timer, train-gate controller, Fischer's mutex

## Key Design Decisions

- Zone (DBM) bounds stored as (value, strict) tuples for proper < vs <= handling
- Floyd-Warshall closure for canonical form (shortest-path)
- Successor: apply guard, reset clocks, check target invariant, time elapse, re-check invariant
- Subsumption: zone1.includes(zone2) prevents redundant exploration
- Fischer's protocol: 3 process states (idle, req, wait, cs) with shared owner variable
  encoded in location names. delta=1 < Delta=2 ensures last-writer-wins mutual exclusion.
